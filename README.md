# instacarAssignment

# Quick Start

```
// Install dependencies

npm install

// Install dependencies client side

npm run client-install

// Run redis server before

// Run the server and Client with concurrently

npm run dev

// Run express server only

npm run start

// Run client only

npm run client

// Run test in server side

npm run test

```
